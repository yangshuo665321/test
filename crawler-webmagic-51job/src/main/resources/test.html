Java开发工程师

1-1.8万/月

深圳市蓝宇飞扬科技有限公司 查看所有职位

深圳-盐田区  |  3-4年经验  |  大专  |  招5人  |  07-31发布  |  计算机科学与技术

五险一金 绩效奖金 年终奖金 交通补贴 定期体检
申请职位 竞争力分析 收藏

职位信息

1. 大专或以上学历，计算机软件或相关专业，3年以上Java实际开发经验；

2. 熟悉Java/JEE， 基础扎实，熟练掌握常用Java技术框架/体系，能编写高质量简洁清晰的代码；

3. 了解各种主流JAVA应用服务器框架，熟练掌握SSH/SSM/SM/SpringMVC框架之一，能完成简单框架配置，具有实际开发经验；

4. 良好的面向对象设计理解，熟悉面向对象设计原则，掌握设计模式及应用场景；

5. 熟悉底层中间件、分布式技术（包括缓存、消息系统、热部署、JMX等），对于高并发、高稳定可用性、高性能、大数据处理有过实际项目产品经验者优先；

6. 熟悉主流项目管理工具（如Project）、数据库建模工具(如PowerDesigner)、设计工具（如Visio）、配置工具SVN等，了解其它主流开发语言、前端页面开发尤佳；

7. 具有较强的责任心、良好的沟通能力、需求理解能力、问题分析和处理能力、学习能力及团队合作精神、喜欢去看及尝试最新的技术，追求编写优雅的代码，从技术趋势和思路上能影响技术团队者尤佳。


职能类别： 软件工程师 互联网软件开发工程师

关键字： SpringMVC JEE 分布式 java

公司信息

  深圳市蓝宇飞扬科技有限公司，是一家以软件与智能设备开发的科技创新型企业。业务范围：智能停车系统（无感支付、扫码支付、场内引导）、人脸识别、生活O2O平台（商家聚合收款，附近商家、用户积分）、聚合支付平台、智能充电桩、智慧社区(社区白条、车位共享、物品共享)、为大中小型企业提供专业的OA、ERP、MRP、HRM等应用设计与开发，以及移动终端UI设计等方面的服务。专业的技术团队、规范的制作流程、完善的售后服务，是我们呈现给企业用户最大的诚意。自始至终，蓝宇飞扬迈坚持秉承以客户价值为依归的经营理念，希望通过更加专业的技术水平和不懈的努力，帮助企业提高运作效率，以及给予指导性的解决方案。
      强大的技术支持是我们企业的核心竞争力，因此在吸引了一支专业研发团队入驻的同时，也非常注重人才专业技能的定期培训和管理，保持专业知识的更新和高效率的步伐，紧跟时代节奏，不断创造出适应市场需求的高效益产品。
